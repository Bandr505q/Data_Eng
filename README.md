## 📂 Repository Structure

This repository contains several Jupyter notebooks with Python practice exercises.

### 📁 Files

### 🟢 First_Home_Work.ipynb
Beginner Python exercises focusing on:
- 🔹 Input / Output
- 🔹 Print formatting
- 🔹 Basic calculations
- 🔹 Simple Python logic

### 🟡 Second_Home_Work.ipynb
String manipulation exercises such as:
- ✏️ Extracting characters from strings
- 🔢 Counting letters, digits, and symbols
- 🔁 Reversing strings
- ✂️ Splitting strings
- 🔍 Working with substrings

### 🔵 Data_Structures.ipynb
Exercises related to Python Data Structures including:
- 🔁 Loops and conditions
- 📋 List operations
- 📦 Tuple operations
- 📚 Dictionary operations

The notebook contains solutions to exercises from the following topics:
- 🔄 Python loops (`for`, `while`, `range`)
- 📋 List manipulation
- 📦 Tuple handling
- 📚 Dictionary operations

---

💡 These notebooks are intended for **learning and practicing Python fundamentals**, especially for beginners.
